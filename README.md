# 躲避他们

一个使用 Godot 4.6 和 C# 开发的 2D 躲避游戏。

## 项目简介

这是一个简单的 2D 躲避游戏，玩家需要控制角色躲避不断生成的敌人，同时尽可能获得更高的分数。

## 功能特性

- 🎮 流畅的 2D 游戏体验
- 🎯 随机生成的敌人和路径
- 📊 实时计分系统
- 🎵 背景音乐和音效
- 🏆 游戏结束界面
- ⌨️ 键盘和手柄支持

## 技术栈

- **游戏引擎**: Godot 4.6
- **编程语言**: C#
- **渲染方法**: GL Compatibility
- **物理引擎**: Godot 内置 2D 物理引擎

## 系统要求

- Godot 4.6 或更高版本
- .NET 8.0 或更高版本
- Windows、macOS 或 Linux 操作系统

## 如何运行

### 使用 Godot 编辑器

1. 下载并安装 [Godot 4.6](https://godotengine.org/download)
2. 打开 Godot 编辑器
3. 点击"导入"按钮，选择项目根目录
4. 点击"运行项目"按钮开始游戏

### 构建可执行文件

1. 在 Godot 编辑器中打开项目
2. 点击"项目" → "导出"
3. 添加预设（Windows、macOS 或 Linux）
4. 点击"导出项目"生成可执行文件

## 游戏控制

### 键盘控制

- **W / 上箭头**: 向上移动
- **S / 下箭头**: 向下移动
- **A / 左箭头**: 向左移动
- **D / 右箭头**: 向右移动
- **空格**: 开始游戏

### 手柄控制

- **左摇杆**: 移动角色
- **A 按钮**: 开始游戏

## 游戏玩法

1. 点击"开始游戏"或按空格键开始
2. 使用方向键或手柄控制角色移动
3. 躲避从屏幕边缘出现的敌人
4. 存活时间越长，得分越高
5. 碰到敌人游戏结束
